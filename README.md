## Vertical Defense Structure (VDS)

This project is a simulation and game built around the concept of a defensive structure set up against incoming threats.

### Overview

The application models a hostile environment where the player must deploy and manage defensive assets to counteract waves of enemies. Core components include projectile mechanics, enemy pathing, and a visual representation of the world and battlefield dynamics.

### Key Modules

- **Game Logic:** Handles the core simulation loop, including enemy spawning, movement, collision detection, and combat states.
- **Entities:** Defines various objects within the world, such as enemies, guns, and projectiles, each with specific behaviors and properties.
- **Visualization:** Manages the drawing of the game world, including background effects (like a star field) and user interface elements (HUD).

This codebase provides a solid foundation for expanding gameplay, implementing specialized defenses, and refining the overall interactive experience.
